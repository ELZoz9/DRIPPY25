# Drippy - Fashion E-Commerce App (Backend)

This repository contains the backend for **Drippy**, a fashion e-commerce mobile application that provides:

- Browsing products
- Product details
- Cart system
- Order creation
- Shipment tracking
- Payments handling
- User authentication with tokens
- Virtual fitting room support
- Seller & admin backend expansion (future)
